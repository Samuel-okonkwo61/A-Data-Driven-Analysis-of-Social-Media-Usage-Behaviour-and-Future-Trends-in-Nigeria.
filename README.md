# A-Data-Driven-Analysis-of-Social-Media-Usage-Behaviour-and-Future-Trends-in-Nigeria.
A Power BI project analyzing social media usage, behavior, and impact in Nigeria. Includes interactive dashboards, KPI insights, and a 10-year forecast model to reveal trends in engagement, productivity, and mental health.


A Data-Driven Analysis of Social Media Usage, Behaviour and Future Trends in Nigeria


Problem Statement
Social media usage in Nigeria has grown rapidly, but its effects on user behavior, productivity, and mental well-being are not fully understood. There is also limited insight into future usage trends and how engagement may evolve over time.
This project aims to analyze current usage patterns, assess their impact, and forecast future trends using data-driven techniques.


Objectives
•	Analyze social media usage across different age groups 
•	Identify the most used platforms 
•	Evaluate the impact on productivity and mental health 
•	Examine the role of social media in income generation 
•	Forecast social media usage over the next 10 years 


Tools & Technologies
•	Power BI – Data visualization and dashboard creation 
•	DAX (Data Analysis Expressions) – Measures and calculations 
•	Microsoft Excel – Data preparation and cleaning 


Key Features
•	Interactive Power BI dashboard with slicers and filters 
•	KPI tracking for user engagement and behavior 
•	Platform usage comparison across demographics 
•	Impact analysis (Productivity & Mental Health) 
•	10-year forecast model using growth rate assumptions 


Key Insights
•	Social media usage is widespread with high daily engagement 
•	X (Twitter), Instagram, and WhatsApp are the most used platforms 
•	Usage is relatively consistent across age groups, with slightly higher engagement among older users 
•	Social media has mixed effects on productivity and mental health 
•	A portion of users leverage social media for income generation 



Forecast
The analysis includes a 10-year forecast model showing a steady increase in social media usage, indicating growing adoption and deeper user engagement in Nigeria.


Conclusion
Social media plays a significant role in communication, business, and daily life in Nigeria. While it offers economic opportunities, it also presents challenges that require balanced and responsible usage.
